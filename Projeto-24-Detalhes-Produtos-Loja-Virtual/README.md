## 📌 Sobre o Projeto

O **Projeto 24** revisita os conceitos de e-commerce com foco total na "Product Page". O objetivo foi criar uma estrutura que suporte descrições detalhadas, galerias de imagens e seletores de opções, mantendo a clareza e a facilidade de navegação tanto em desktop quanto em dispositivos móveis.

### ⚙️ Funcionalidades e Conceitos Aplicados:
- **Hierarquia de Informações:** Organização estratégica de títulos, preços, descrições e botões de chamada para ação (CTA).
- **Galeria de Imagens Responsiva:** Estruturação para exibição de múltiplas fotos do produto, essencial para a decisão de compra.
- **Micro-interações:** Uso de CSS para feedbacks visuais em botões, seletores de quantidade e estados de hover.
- **Layout Adaptável:** Garantia de que a página de detalhes mantenha a funcionalidade e estética em telas de diferentes proporções.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica para garantir SEO e acessibilidade nos detalhes do produto.
- **CSS3:** Estilização focada em componentes modernos de e-commerce e grid layout.
- **Google Fonts:** Tipografia selecionada para aumentar a legibilidade das descrições.

## 🧠 Destaque Técnico: Foco na Conversão

O design desta página foi pensado para reduzir a fricção do usuário, utilizando técnicas de posicionamento que mantêm as informações cruciais sempre visíveis:
```CSS

/* Exemplo de destaque para o botão de compra */
.btn-compra {
    background-color: #ff9900;
    color: #fff;
    padding: 15px 30px;
    font-weight: bold;
    border-radius: 5px;
    transition: transform 0.2s;
}

.btn-compra:hover {
    transform: scale(1.05);
    background-color: #e68a00;
}
``
