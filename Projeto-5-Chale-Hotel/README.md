## 📌 Sobre o Projeto

O **Chalé Hotel** é um projeto de Landing Page institucional que explora a harmonia entre imagens de alta qualidade e espaços negativos. O objetivo foi criar uma interface que transmita tranquilidade e profissionalismo, utilizando técnicas de posicionamento CSS para organizar galerias de fotos e formulários de contato de maneira integrada.

### ⚙️ Funcionalidades e Conceitos Aplicados:
- **Design Minimalista:** Uso de uma paleta de cores em tons de bege, azul e branco para reforçar a identidade visual do setor hoteleiro.
- **Layout Centralizado e Flexível:** Estruturação de containers que garantem que o conteúdo permaneça focado e legível.
- **Estilização de Navegação Dinâmica:** Menu superior com estados visuais que indicam a página ativa e interações de hover.
- **Galeria de Serviços:** Organização de cards de informações sobre acomodações, lazer e gastronomia.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Marcação semântica para cabeçalhos, formulários e seções de conteúdo.
- **CSS3:** Gerenciamento de backgrounds complexos, bordas arredondadas e tipografia personalizada via `@font-face` ou Google Fonts.

## 🚀 Como Executar

1. Clone o repositório:
    ```Bash

    git clone [https://github.com/jaype10/Treino-Web.git](https://github.com/jaype10/Treino-Web.git)
    ```

2. Navegue até a pasta:
    ```Bash

    cd Treino-Web/Projeto-5-Chale-Hotel
    ```

3. Abra o arquivo `index.html` no seu navegador.
